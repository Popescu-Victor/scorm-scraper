I've started this project to deal with lack of access to proper data analytics tools on my and my colleague's work laptops. Working as a LMS Administrator in the military, I'm highly limited when it comes to what I can and what I cannot install on my work computer and my coworkers who live in other cities even more so. Therefore I've created this tool in order to make the reporting process much more efficient. The NCA tool has two different version at the moment, the one built on a TKinter interface for computers that can run .exe files or IDEs (my own has Thonny on it while VScode and Neovim are blocked by the security policy) and another version built on a Gradio interface which can run in Google Colab.
